## Featured Projects

### 📝 TOEFL & TOEIC Prediction Test System
*Global Media Cendekia* | June 2023 — September 2023

[**exam.gmcendekia.com**](https://exam.gmcendekia.com)

A full-stack score prediction platform that helps students assess their English proficiency readiness for TOEFL and TOEIC exams.

- Built a responsive test-taking interface using **Next.js**, **React.js**, **Shadcn UI**, and **Radix UI**
- Developed robust backend API with **AdonisJS** and **PostgreSQL** for test sessions, scoring logic, and result tracking
- Integrated **Cloudflare R2** (S3-compatible object storage) for scalable file uploads
- Provisioned and managed a **Linux-based VPS** from scratch, including server hardening and application deployment

*Tools: Next.js, React.js, Shadcn UI, Radix UI, AdonisJS, PostgreSQL, Cloudflare R2, Linux VPS*
